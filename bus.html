OneBusAway

Google Material Design

Although Onebusaway provides real time information for public transit, users need to spend a lot of time on completing the tasks ineffectively. In addition, as a daily app user, I found there are features that are accessible in less obvious places and hence seldom used. This self-initiated project aims to obviate usability issues and explore new possibilities through a user interface redesign for the current OneBusAway app to ensure it provides an intuitive and streamlined user experience.


ROLE

UI/UX Designer

METHODS

Ideation, Persona, Flow Analysis, Mockups

DURATION

Aug 2015


Design Process

  • Define goal and scope of the project.
  • Identify the usability issues.
  • Create personas and case scenarios.
  • Ideate on design options and simplify the flow.
  • Craft mockups to demonstrate user flow and interactions.

    • Goal

  • Reconstruct app navigation structure.
  • Reduce number of clicks of completing a task.
  • Simplify the UI and refine the visual hierarchy.
  • Promote features that support new user cases.
  • Apply Google material design language.
  • Keep as much consistency as possible with the existing brand.
